# ML Service Platform

Сервис машинного обучения с личным кабинетом пользователя.

## Функциональность
- Регистрация и авторизация пользователей
- Пополнение баланса (кредитная система)
- Выполнение ML предсказаний
- История транзакций и предсказаний
- Валидация входных данных
- Администраторская панель
